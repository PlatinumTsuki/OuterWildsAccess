# Changelog

## 1.0.0 (2026-03-03)

First stable release.

### Features
- Screen reader support (NVDA, JAWS) for all menus, dialogue, game state, and death causes
- Audio navigation with 6 categories: Ship, NPCs, Interactables, Nomai Texts, Locations, Signs
- Auto-walk with A* pathfinding, obstacle avoidance, and automatic jumping
- Audio guidance with 5-tier tonal alignment cues
- Ship autopilot to 13 destinations
- Ship recall (teleport ship to player)
- Nomai text instant reading (bypasses translation animation)
- Ship log reader accessible from anywhere
- Flight telemetry (speed, altitude, orientation)
- Personal status (H), ship status (J), environment status (K), detailed position (L)
- Loop timer (F2)
- Low resource alerts (automatic)
- Obstacle collision beeps
- In-game help menu (F1) with 6 categories
- Accessibility settings menu (F6) with 13 toggles
- Meditation available from the start
- Ghost matter protection
- Peaceful ghosts (DLC, optional)
- Bilingual: French and English (auto-detected from game language)
- NVDA direct API support with speech priority system
