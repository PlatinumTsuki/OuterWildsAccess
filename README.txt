OuterWildsAccess v1.0.0-beta
=============================
Accessibility mod for Outer Wilds — screen reader support for blind and
visually impaired players. Works with NVDA, JAWS, and other screen readers.

REQUIREMENTS
- Outer Wilds (Steam or Epic)
- OWML 2.15.5+ (Outer Wilds Mod Loader)
- A screen reader (NVDA recommended)

INSTALLATION
1. Install OWML if not already done:
   https://outerwildsmods.com/

2. Copy the "OuterWildsAccess" folder into:
   [Your Game Folder]\OWML\Mods\

3. Copy these two files to the GAME ROOT folder
   (same folder as OuterWilds.exe):
   - Tolk.dll
   - nvdaControllerClient64.dll

4. Launch the game via OWML.

FEATURES
- Screen reader announcements for menus, dialogue, game state, death causes
- Audio navigation with 6 categories (ship, NPCs, interactables, Nomai texts, locations, signs)
- Auto-walk with A* pathfinding
- Audio guidance with tone cues
- Ship autopilot to any planet
- Ship recall, meditation unlock, ghost matter protection
- Loop timer, ship/personal/environment status
- Signalscope and Nomai text reading
- In-game help (F1) and settings menu (F6)
- Bilingual: French and English (auto-detected from game language)

CONTROLS
Press F1 in-game for the full help menu.
